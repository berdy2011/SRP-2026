# SentrySite AI

## 1. Название проекта
**SentrySite AI** — ИИ-ассистент для выбора безопасных мест под строительство.

## 2. Цель проекта
Помочь инженерам выбирать локации для школ, больниц и мостов,
анализируя качество воздуха (PM2.5) и состояние грунта.

## 3. Какую проблему решает
Сейчас инженеры проверяют экологию и геологию по отдельности.
Это долго и неудобно. SentrySite AI объединяет эти данные
и выдаёт понятный отчёт — безопасно здесь строить или нет.

## 4. Ключевые функции
* **Оценка безопасности:** проверяет координаты на соответствие
  норме ВОЗ по PM2.5 (5 мкг/м³).
* **Пресеты по типу объекта:** для школ — акцент на здоровье детей,
  для мостов — на прочность материалов и грунт.
* **ИИ-отчёт:** превращает сухие цифры в читаемые рекомендации
  для инженера.
