# SentrySite AI

## 1. Название проекта
**SentrySite AI** — интеллектуальный гео-экологический ассистент для гражданских инженеров и градостроителей.

## 2. Цель проекта
Автоматизировать комплексный анализ качества воздуха ($PM_{2.5}$) и параметров грунта для выбора наиболее безопасных и долговечных локаций при строительстве объектов критической инфраструктуры (школ, больниц, мостов).

## 3. Какую проблему решает
При планировании социально значимых объектов инженеры часто анализируют геологические и экологические данные раздельно. Размещение школ или больниц в зонах с высоким уровнем мелкодисперсных частиц ($PM_{2.5}$) создаёт долгосрочные риски для здоровья людей. В то же время, агрессивные химические примеси в воздухе и нестабильный грунт ускоряют износ конструкций (например, мостов). Инструмент переводит сложные экологические индексы в понятные инженерные рекомендации.

## 4. Ключевые функции
* **Режим оценки безопасности (Safety Evaluation):** Проверка координат объекта на соответствие жестким экологическим стандартам (базовый лимит ВОЗ — 5 мкг/м³).
* **Контекстные пресеты под тип объекта:** Изменение логики анализа в зависимости от здания (для школ — приоритет на детское здоровье и фильтрацию воздуха; для мостов — на коррозийную стойкость материалов и устойчивость грунта).
* **Интеграция с LLM (Oylan API):** Преобразование сухих табличных данных и индексов NowCast AQI в развернутые текстовые отчеты и рекомендации для инженеров.
